# Tachyon
### A Instanciable Server Framework in Kotlin

## Introduction
A Tachyon is an unit instaciable server there are three types of Tachyons

| Tachyon    	| Nature       	|
|------------	|--------------	|
| Single     	| Blocking     	|
| Serial     	| Blocking     	|
| Concurrent 	| Non Blocking 	|
